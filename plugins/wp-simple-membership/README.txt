Simple Wordpress Membership plugin. Front End login, Register, and My profile,

== Description ==

This plugin is used to create a wordpress Membership Website. Shortcodes can be used to create Login, Registration and My Profile page.

<a href="http://www.janothemes.com/wp-simple-membership/">Demo And Docs</a>

features :

<ul>
<li>Front end login page.</li>
<li>Front end Registration page.</li>
<li>Front end my profile page.</li>
<li>Front end Reset password page.</li>
<li>Redirect to a page of your choice after login.(Adminisrator will still reidrect to admin dashboard.)</li>
<li>Redirect to a page of you choice after user registration. Like thank-you page.</li>
<li>Send New user notification to admin only or user only or both or none. </li>
<li>Alot of actions and filters supported to change the behaviour like Form fields..</li>
</ul>

== Installation ==
1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `Wp simple Membership` from Plugins page
